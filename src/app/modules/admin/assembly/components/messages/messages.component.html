Messages

Introduction

asm-message is a highly customizable and detailed component for displaying all kinds of messages across your app. They can be controlled from your components as well as via the AsmMessageService.

Module

import { AsmMessageModule } from '@assembly';

Appearance

Icons

The info, success, warning and error type message boxes have default icons and they are visible by default. If you wish to hide those icons, use the [showIcon]="false" input.

Example
Success message with no icon

Custom icons

All message boxes come with custom icon support. To use a custom icon, simply use asmMessageIcon attribute on an element you wish to use as the icon. If there is a default icon for the message box, custom icon will override it.

Example
Info message with a custom icon

Dismissible messages

Dismissible messages can be removed from the view by clicking their X button. In order to get the dismiss button visible, the [dismissed] input must be explicitly set either to true or false. Setting the [dismissed] input to null or simply removing it from the element will remove the X button.
