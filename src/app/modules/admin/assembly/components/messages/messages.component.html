Messages

Introduction

asm-message is a highly customizable and detailed component for displaying all kinds of messages across your app. They can be controlled from your components as well as via the AsmMessageService.

Appearances

Message boxes have 3 different built-in appearances to choose from. You can use them as they are or modify them for your own needs and likings.

Dismissible boxes

Dismissible boxes can be removed from the view by clicking their X button.

Example
Your subscription model is successfully upgraded to the Lifetime Pro. This one has a lot of text so you can see how the text is going to be displayed. Also this is a dismissible message box which can be dismissed by clicking the close button.
