Messages

Introduction

asm-message is a highly customizable and detailed component for displaying all kinds of messages across your app. They can be controlled from your components as well as via the AsmMessageService.

Module

import { AsmMessageModule } from '@assembly';

Appearance

Message boxes have 2 different built-in appearances to choose from. You can use them as they are or modify them for your own needs and likings.

Solid (default)
Primary message box Accent message box Warn message box Basic message box Info message box Success message box Warning message box Error message box
Outline
Primary message box Accent message box Warn message box Basic message box Info message box Success message box Warning message box Error message box

Icons

The info, success, warning and error type message boxes have default icons and they are visible by default. If you wish to hide those icons, use the [showIcon]="false" attribute.

Example
Success message with no icon

Custom icons

All message boxes come with icon support. To use a custom icon, set the [customIcon]="true" and use asmMessageIcon attribute on an element you wish to use as the icon.

Example
notifications Info message with a custom icon

Complex content

The content of the message box can also be set using the asmMessageContent attribute which may allow to have a better structure on complex contents.
