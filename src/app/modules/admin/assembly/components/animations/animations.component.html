Animations

Assembly provides set of ready-to-use animations for convenience.

Expand / Collapse

Current state: {{animationStates.expandCollapse}}
Animated box
Preview
Expand / Collapse animation can be triggered in two ways;
  1. Adding [@expandCollapse]="state" to the element and toggling the state between the 'collapsed' and 'expanded' strings manually.
  2. Adding @expandCollapse to the element and using *ngIf to toggle the element.
