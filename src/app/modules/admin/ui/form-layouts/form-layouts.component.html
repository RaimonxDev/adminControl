UI ELEMENTS / FORM LAYOUTS

Form Layouts

Forms are one of the most important and most used components of any web application. Below, you can find lots of ready to use form layout examples which you can modify for your needs and immediately start coding your logic.

These examples uses TailwindCSS helper classes. We strongly recommend that you use TailwindCSS as well to build your form layouts. This not only allows an extreme flexibility on your forms but it also saves time since forms are more generic components and they don't necessarily require explicit id/class naming on the form or structural element when using them with Angular because of how Angular handles the forms.
