UI ELEMENTS / MATERIAL COMPONENTS

Overview

Introduction

Assembly uses Angular Material as its primary user interface library. It offers form controls, buttons, tabs, sidebars, icons, modals, tooltips, data tables and many more well tested and widely used components.

Even though Assembly doesn't follow Google's Material design specifications, Angular Material is one of the best Angular component libraries out there and since it's developed and maintained by Google itself, it offers the best compatibility and support for Angular.

Assembly is 100% compatible with all Angular Material components.

Components

Under Components section, you can find all the base components that Angular Material library provides. Examples and documentation on the Components section are embedded directly from the Angular Material's official documentation and they are for the Angular Material version which Assembly currently uses. It also provides a visual testing bed for Assembly to test the compatibility of the components.

Please note, the Components section may be couple versions behind of the latest version of Angular Material. Even though Assembly updates regularly, it's not possible to release updates as soon as Angular or Angular Material releases a new version.
