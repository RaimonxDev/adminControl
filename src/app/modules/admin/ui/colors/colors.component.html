UI ELEMENTS / COLORS

Colors

Introduction

Assembly provides set of ready to use colors for convenience. These colors based on Google's Material Design specs. and each color comes with a set of class names that can be added to an element to change its text and background colors separately as well as both of them at the same time.

Usage

Base classes

Each color class has 3 different variants;

  • .COLORNAME

    Adds the specified color as the background and its contrasting color as the text color.

  • .bg-COLORNAME

    Adds the specified color as the background color.

  • .text-COLORNAME

    Adds the specified color as the text color.

Hue levels

There are different hue values available for all colors except the White and Black palettes. To access the certain hue value of the color, simply add the value at the end of the class name:

  • .COLORNAME-HUE
  • .bg-COLORNAME-HUE
  • .text-COLORNAME-HUE
Example
Cyan (600) background & contrasting color
Cyan (400) background
Cyan (300) color

Emphasize helpers

Following class names also available to have different emphasize levels on the texts:

  • .text-secondary
  • .text-hint or .text-disabled
  • .text-fade or .divider
Example

Primary, Accent, Warn, White & Black

Primary, Accent and Warn palettes will use the colors from the Angular Material's color theme making it easier to access theme colors to have consistency throughout the application. They will change depending on the color theme of your app.

White and Black palettes are fixed and cannot be modified.
