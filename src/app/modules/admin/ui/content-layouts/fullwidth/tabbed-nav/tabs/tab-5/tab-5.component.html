Fifth Tab Content

Tabbed nav layout features a tab-like navigation that works with Router. Clicking on a tab will trigger a route change and the active tab will also change accordingly. Watch the browser's url bar as you click on the tabs to see it in action. Manually navigating to the url also works.

This type of layout is especially useful if you need a tab-like sub navigation within your page/app without having to put everything into single component like normal tabs.
